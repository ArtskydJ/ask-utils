# Utility functions for ask-sdk

## Getting started

```
$ npm i -S ask-utils
```

## Usage

```
const { randomResponse, intentHandlers } = require('ask-utils')
```

## development

```
$ git clone git@github.com:hideokamoto/ask-utils.git
$ cd ask-utils
$ npm i
```

### test

```
$ npm test
```

### Lint

```
$ npm run lint

or

$ npm run lint -- --fix
```
